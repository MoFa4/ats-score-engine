# ATS Score Engine

🚀 A simple, anime-themed web app to calculate ATS (Applicant Tracking System) score for any job role.  

The app allows users to:

- Upload a **resume** (PDF or text).  
- Enter a **job description**.  
- **Automatically extract skills** from both documents.  
- Calculate a **predicted score** and **chances of getting shortlisted**.  
- Highlight **matched skills**, **missing skills**, and **extra skills**.  

This app works for **all types of roles**: tech, non-tech, hardware, BPO, creative, and more.

---

## **Features**

- Auto skill extraction from resume and job description.  
- Supports 500+ common skills across multiple domains.  
- Anime-inspired UI for a unique and colorful interface.  
- Quick, intuitive, and fully interactive.  
- Results update only after clicking **Calculate**, and reset on refresh.
